# Rick And Morty
____
Пет-проект. Приложение реализует часть API Rick And Morty (персонажи).
____
## Стек:
- Swift
- UIKit
- MVVM
- RxSwift
- Kingfisher
____
## Локализация
- Русский язык (частично)
- Английский язык
